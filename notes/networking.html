The OSI Model (Bottom-Up)

1. Physical → 2. Data Link → 3. Network (IP) → 4. Transport (TCP/UDP) → 5. Session → 6. Presentation → 7. Application (HTTP)

Essential CLI Tools

ip addr show # Check local IP addresses
dig google.com # DNS lookup information
traceroute 8.8.8.8 # Trace path to a destination
nc -zv 127.0.0.1 80 # Check if port 80 is open (Netcat)
