Container Lifecycle

docker build -t app:v1 . # Build image
docker run -d -p 80:80 app # Run detached
docker ps # List running containers
docker exec -it [ID] sh # Enter container shell

Dockerfile Best Practices

  • ✅ Use Alpine images to keep size small.
  • ✅ Use .dockerignore to exclude node_modules/git.
  • ✅ Combine RUN commands to reduce layers.
