Kubectl Power User Commands

kubectl get pods -A # List all pods in all namespaces
kubectl apply -f deployment.yaml # Deploy resource
kubectl logs -f [pod-name] # Stream logs
kubectl describe node [name] # Debug node health

Pro Tip: Use alias k='kubectl' to save time during production outages!
